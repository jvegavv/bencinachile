Getting started

Installation and Setup

Resideo template is pure HTML and it doesn't require any special installation process. You can just upload it on a server or open it locally and it works.

Themes

The template has two color tone variations, or modes: light and dark. Each version has its own folder which can be used independently.

Google Maps

The following pages 

contact.html
properties.html
single-property.html
submit-property.html

are using Google Maps and in order to use them Google requires an API key. You can get it from here. After getting the key, open the above files and replace YOUR_GOOGLE_MAPS_KEY_HERE text with your key from this line at the end of the file:

<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_GOOGLE_MAPS_KEY_HERE&amp;libraries=geometry&amp;libraries=places" type="text/javascript"></script>

CSS

The template is based on Bootstrap v4.3.1. For Bootstrap css and components, please check here.

All custom css styles are found in /css/style.css file.

Icons

The template makes use of Font Awesome 4.7.0 web font as its icons set.

Modes

The template has two modes: light and dark. It's simple to switch between them just by adding and removing the .pxp-dark-mode CSS class to the body HTML tag. Also you have to make sure when using the dark mode to add .text-white CSS class to the text elements that are being displayed on a dark background.

Javascript

Files

The template client-side functionality is based on jQuery javascript library.

Javascript custom files included:

  • main.js — Main interactions functionality
  • gallery.js — Gallery functionality for single property page
  • map.js — Map functionality for properties list page
  • single-map.js — Map functionality for single property page
  • submit-property-map.js — Map functionality for submit property page
  • contact-map.js — Map functionality for contact us page

Libraries

External javascript libraries included:

  • Chart.js
  • DropzoneJS
  • Infobox
  • jQuery Sticky
  • Markerclusterer
  • Owl Carousel
  • PhotoSwipe

Google Maps

Google Maps API (more on how to use the API here) is used on the following pages:

  1. properties.html

    The functionality for the map is found in map.js file. The properties markers displayed on the map are generated from JSON format, which is defined inside the map.js file, like this:

    propertiesList = [{
    id: 1,
    title: 'Chic Apartment in Downtown',
    photo: 'images/ph-thmb.jpg',
    position: {
        lat: '37.779023',
        lng: '-122.48862200000002'
    },
    price: {
        long: '$890,000',
        short: '$890k'
    },
    link: 'single-property.html',
    features: {
        beds: 2,
        baths: 2,
        size: '920 SF'
    }
}, { 
    ... 
}];

  2. single-property.html

    The functionality for the map is found in single-map.js file. The marker coordinates are given by these two variables:

    var propLat = '37.75347110977809';
var propLng = '-122.46686778459474';

  3. submit-property.html

    The functionality for the map is found in submit-property-map.js file. The marker coordinates are given by these two variables:

    var propLat = '37.75347110977809';
var propLng = '-122.46686778459474';

  4. contact.html

    The functionality for the map is found in contact-map.js file. The marker coordinates are given by these two variables:

    var propLat = '37.75347110977809';
var propLng = '-122.46686778459474';
